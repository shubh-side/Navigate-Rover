Instructions

    This Game is all about navigating the Mars Rover.
  • 1. Green box in th grid depicts the location of the Rover.
  • 2. Red box depicts the destination station where you need to direct the Rover.
  • 3. By clicking you can place the walls between the Rover and Destination station.
  • 4. After that you can customize the way Rover will search the whole space grid like whether search with Bi-Directionality
    or with Diagonal Moves etc you will find in the Custom Panel placed in left.
  • 5. After setting up the customs then you can hit the start button and the Rover will provide you the shortest path between Rover
    and Destination station.
